# Robot War Simulator

## Description

Robot War Simulator is a C++ battlefield simulation project developed for the course CCP6124 Object Oriented Programming and Data Structures. The project simulates a turn-based robot battle environment where different robot types with unique abilities compete against each other on a battlefield.

The simulator applies Object-Oriented Programming concepts such as inheritance, polymorphism, encapsulation, and abstraction to create modular and extensible robot behaviors. Each robot type is designed with its own special abilities, movement patterns, and attack strategies.

The system includes battlefield management, robot interaction, movement and attack mechanics, health and respawn systems, and battle logging. Simulation settings can be configured through input files, allowing different battlefield sizes, robot combinations, and simulation steps.

This project was developed for educational purposes to strengthen understanding of C++ programming, data structures, and OOP design principles.
